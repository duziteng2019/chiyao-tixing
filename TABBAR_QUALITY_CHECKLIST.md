## 🐛 已知问题与解决方案

### 常见问题排查

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|---------|
| 图标不显示 | 文件路径错误 | 检查app.json中的iconPath |
| 图标显示过大/过小 | 尺寸不符合规范 | 重新导出为81×81px或162×162px |
| 图标有白底 | 未保留透明通道 | 检查SVG背景设置和导出选项 |
| 选中后颜色不变 | selectedColor配置错误 | 更新app.json中的selectedColor值 |
| 图标模糊 | 分辨率不足 | 使用@2x或@3x版本 |
| 部分设备不兼容 | 色彩空间问题 | 确保使用sRGB模式 |

### 回滚方案

如果新图标系统出现问题：

```bash
# 备份当前版本
cp images/*.png images/backup/

# 如需回滚到旧图标
# (假设有旧版本备份)
cp images/backup-old/*.png images/
```

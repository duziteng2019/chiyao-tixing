# 吃药提醒小程序

一款功能完整的微信小程序，帮助用户管理药品信息、设置服药提醒、追踪服药记录。

## 功能特性

- 用户注册与登录系统
- 药品信息管理（添加、编辑、删除药品）
- 自定义提醒设置（支持多种提醒模式）
- 提醒通知功能
- 服药记录追踪
- 数据统计与分析
- 本地存储与云端同步

## 项目结构

```
吃药提醒小程序/
├── pages/                  # 页面文件
│   ├── index/             # 首页
│   ├── medication/        # 药品管理
│   ├── reminder/          # 提醒设置
│   ├── records/           # 服药记录
│   ├── profile/           # 个人中心
│   ├── add-medication/    # 添加药品
│   └── add-reminder/      # 添加提醒
├── utils/                 # 工具函数
├── images/                # 图片资源
├── cloudfunctions/        # 云函数
├── app.js                 # 小程序入口
├── app.json               # 小程序配置
├── app.wxss               # 全局样式
└── sitemap.json           # 站点地图
```

## 开发说明

1. 在微信开发者工具中打开项目
2. 配置云开发环境ID（修改 app.js 中的 env 配置）
3. 在云开发控制台创建数据库集合：
   - users（用户信息）
   - medications（药品信息）
   - reminders（提醒配置）
   - records（服药记录）
4. 部署云函数

## 使用说明

1. 首次使用需要登录
2. 添加药品信息
3. 为药品设置提醒
4. 收到提醒后记录服药状态
5. 查看服药统计数据

## 技术栈

- 微信小程序原生开发
- 微信云开发
- 本地存储（wx.setStorageSync/wx.getStorageSync）

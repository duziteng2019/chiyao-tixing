## 🔧 方法二：命令行自动部署（进阶）

### 前置条件：
1. 下载CI密钥文件（从微信公众平台）
2. 重命名为 `private.key` 放到项目根目录
3. 安装依赖：`npm install --prefix . deploy-package.json`

### 执行命令：
```bash
# 部署单个云函数
node deploy-cloud-function.js sendReminder

# 部署所有云函数
node deploy-cloud-function.js all
```

## ⚠️ 常见问题解决

### Q1: 右键菜单没有"上传"选项？
**原因：** 未关联云开发环境  
**解决：** 
- 在 app.js 中确认 env 配置正确
- 或在开发者工具中点击「云开发」按钮开通

### Q2: 提示"依赖安装失败"？
**原因：** package.json 格式错误或网络问题  
**解决：**
- 检查 cloudfunctions/sendReminder/package.json
- 确保有网络连接
- 重试上传操作

### Q3: 部署成功但收不到消息？
**排查清单：**
- [ ] 用户是否点击了「允许」订阅
- [ ] 模板字段是否匹配（thing1, time2, thing3）
- [ ] 是否到达设定的提醒时间
- [ ] 查看云函数日志确认是否被调用
